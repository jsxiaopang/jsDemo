1. 移动端横向滚动条 隐藏

移动端使用原生滚动条 要加-webkit-overflow-scrolling: touch。 一般可以加在body上面，这样很方便。 不过这样可能会出现问题， 闪屏。。 所以最好加在需要的dom上。

我们如果想用原生滚动条，并且想隐藏滚动条，需要设置 滚动的dom高度比父级高11px，然后父级元素overflow-y:hidden, 这样就可以把滚动条给覆盖掉。或者可以父级元素margin-bottom:-20px; padding-bottom:20px; 效果一样。

2. 移动端局部滚动条 常见问题

这种常见的是 局部滚动条， 使用原生局部滚动条的时候 滑动的时候 会触发全局滚动， 并且有时候触发不了。针对这种情况， 有些人选择使用框架 iscroll， 我还是比较喜欢原生的。 解决这个问题。 只需要用 ScrollFix，github即可。 其实就是在滑动到头部，或者底部的时候， js给 局部滚动scrollTop 设置为1 or 局部滚动的区域 高度－1.

var ScrollFix = function(elem) { // Variables to track inputs var startY, startTopScroll; elem = document.querySelector(elem); // If there is no element, then do nothing if(!elem) return; // Handle the start of interactions elem.addEventListener('touchstart', function(event){ startY = event.touches[0].pageY; startTopScroll = elem.scrollTop; if(startTopScroll <= 0) elem.scrollTop = 1; if(startTopScroll + elem.offsetHeight >= elem.scrollHeight) elem.scrollTop = elem.scrollHeight - elem.offsetHeight - 1; }, false); };
